# Cats Appear! ≽^•⩊•^≼

![Demo](https://github.com/k-wong/cats-appear/blob/main/screenshot-1280x800.png)

Cats Appear! is a Chrome extension that shows a random cat gif on the active tab at random intervals.

## Features

- Displays a random cat gif every 2-10 minutes.
- Summon a cat gif immediately by clicking the extension icon.
- Does not collect, transmit, or store user data.

## Local Installation

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this project folder.

## Permissions

- `alarms`: schedules the next random cat appearance.
- `scripting`: injects the local overlay script into the active tab.
- `<all_urls>`: allows the cat overlay to appear on normal webpages.
